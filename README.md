# MJC Kitchen Design Solutions — Landing Page

Premium landing page for MJC Cabinets (Orlando, FL) optimized for Meta Ads and Google Ads traffic.

## Stack
- Vanilla HTML + CSS + JS (single file, zero dependencies)
- Google Fonts (Playfair Display + DM Sans)
- Schema.org LocalBusiness markup
- Mobile-first responsive

## Deploy
Static site — deploy anywhere (Vercel, Netlify, Cloudflare Pages, S3).

**Vercel:** Import this repo, no build step needed, output directory is root.

## Before going live
1. Add project images to `/images/` folder (`project-01.jpg` to `project-10.jpg`)
2. Connect the form to a webhook (n8n, Typeform, Formspree) — see `lead-form` handler in `index.html`
3. Paste Meta Pixel ID in the head (search for `PIXEL_ID`)
4. Paste Google Ads conversion tag (search for `AW-XXXXXXXX`)
